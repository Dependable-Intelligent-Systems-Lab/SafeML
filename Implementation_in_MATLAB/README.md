## Wasserstein Distance
Simple example for Wasserstein distance measure function in MATLAB:

<pre>X = normrnd(1,1,[100,1]);
Y = normrnd(1,1.1,[100,1]);
WS_Dist = Wasserstein_Dist(X,Y)</pre>
